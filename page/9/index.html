关于设置APP启动图有两种方式
1.旧方式：

Launch Images Source 添加启动图，Launch Screen File 中删掉“LaunchScreen”,置为空；

在Assets文件夹里新建一个iOS Launch Image。然后设置好对应的尺寸。
LaunchScreen.storyboard取消勾选LaunchScreen。


2.新方式：
Launch Images Source 中不需要添加启动图，Launch Screen File 为“ LaunchScreen”（默认）
，LaunchScreen.storyboard添加imageView,imageView上放图片，工程中要有对应的 7501334 的@2x图，1242 2208的@3x图 ，LaunchScreen.storyboard勾选LaunchScreen。

如果你的模拟器可以加载启动图个，真机启动图加载不来，那么把 7501334 的@2x图，1242 2208的@3x图 命名为Defult即可。
